# DPD AI Resume Builder 🚀

[![](https://img.shields.io/badge/Live-Demo-blue?style=for-the-badge&logo=vercel)](https://dpdai-resume.vercel.app)

## 📊 Project Overview
As a professional with 14 years of experience, I engineered this tool to solve the "ATS Black-Hole" problem. The application uses advanced semantic parsing to extract unstructured data and map it into optimized, single-column layouts that modern Applicant Tracking Systems (Workday, Greenhouse, Taleo) can process with 100% accuracy.

## 🛠 Tech Stack
*   **Frontend**: HTML5, Tailwind CSS, JavaScript
*   **Data Processing**: PDF.js, Automated JSON Parsing
*   **AI Integration**: Google Gemini API for semantic text optimization
*   **Analytics**: Integrated live ATS-scoring engine
